# 



## Installation

```console
$ npm install textarea-no-scroll
```
## Usage

Example:

```console
 import TextareaNoScroll from 'textarea-no-scroll';
 
 ReactDOM.render(
     <TextAreaNoScroll />, 
     document.getElementById('app')
 );
 
```

## Props
```
    className: string,
    disabled: string,
    value: string,
    defaultValue: string,
    placeholder: string,
    onChange: func,
    inputRef: func,
    maxLength: number, // 字数长度
    shouldHideLimitReset: bool, // 隐藏字数

```
